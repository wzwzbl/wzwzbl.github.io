关闭selinux ： setenforce 0
selinux 三种模式
1.enforcing 强制模式 1
2.permissive 宽松模式 0
3.disabled 关闭模式
关闭防火墙：firewall-cmd –set-default-zone=trusted
systemctl stop firewalld
变量
环境变量env
set观察所有变量
PS1 提示字符的设置
列如PS1=’[\u@\h\w\A##]$‘
\u目前用户的账号名称。
\h
\w完整的工作名称
\A显示时间，为24小时格式。
#执行的第几个命令
$ 提示符

export 也是变量的
locale 语系变量
修改：cat /etc/locale.conf

declare 声明变量类型
declare -i sum=10+10 声明整数变量
echo ${sum}
20
declare -x sum 声明环境变量
export | grep sum

-a数组，-r不能被更改
ulimit 文件大小限制
通配符
*代表任意个字符
？代表一个字符
【0-9】，代表0-9
^取反

&&与 只有前面正确，才会进行后面的操作
|| 或 只有前面错误，才会今后后面操作

cut 主要作用是将同一行的数据进行分解
列：echo ${PATH} | cut -d ‘:’ -f 3,5

grep 过滤查找 -v取反 -n 显示行号

last 列出目前和过去登入系统用户的相关信息。

sort 排序 -t ‘:’ 分隔符
-k 用第几排排序
uniq 将重复的列出显示
wc计数

字符转换符 tr -d 删除 -s替换
col 将文件里的 tab 换成 空格键
join 将相同数据的那行加在一起
paste 直接将俩行贴在一起
expand 将tab 转换成空格键
划分命令split 将大的文件划为小的文件
参数代换xargs 产生某个命令的参数
